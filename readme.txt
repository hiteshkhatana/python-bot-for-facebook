                                         ** Python bot for facebook **

Instructions :-

(1) Install dependancies required using requirements.txt file.

(2) Download chromedriver which supports your chrome browser from  https://chromedriver.chromium.org/downloads

(3) There are four functions for different tasks - 
	a. Login , 
	b. Adding friend from same location ,
	c.  Updating status and 
	d. commenting on random friend's recent post.

(4) Program will take input for performing certain actions.
	A for adding friend
	S for updating status
	C for commenting

(5) Enter your email and password to login.

(6) You have to enter friend's name to add from same location , Status for updating status , comment for commenting on friend's post.

(6) To exit program and close the driver ,press Q.
